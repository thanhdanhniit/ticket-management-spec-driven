# Secondary Settings Sidebar

## Purpose
Provides dedicated, localized navigation for all configuration and administrative modules nested under the main "Settings" area of the application.

## Layout
A vertical, fixed-width panel located on the left side of the screen, immediately adjacent to the Global Sidebar.
- **Header Section:** A large, prominent title ("Settings") indicating the current global context.
- **Body Section (Scrollable):** A list of navigation links

## Components
- List Items / Navigation Links
- Active State Indicators (Highlighting the currently selected page)

## Fields
- **Title:** "Settings"
- **Category Header:** "Users" (rendered as smaller, muted, upper-case or distinct typography).
- **Navigation Items (Under Users):**
  - Permissions
  - Teams
  - Postmortem Templates
  - Audit Logs

## User Actions
- Navigate between different administrative configurations by clicking the respective links.
- Visually identify the currently active settings module via its distinct active state styling (e.g., bordered or background-highlighted text block).
